Résumé des Classes Layout

Grille et Flexbox
  • container - Conteneur de base
  • container-fluid - Conteneur fluide
  • row - Ligne de grille
  • col - Colonne de base
  • col-* - Colonne avec largeur
  • offset-* - Décalage de colonne
  • d-flex - Affichage flex
  • justify-content-* - Justification
  • align-items-* - Alignement
Espacement et autres
  • m-* - Marges
  • p-* - Padding
  • mt-*, mb-* - Marges verticales
  • ms-*, me-* - Marges horizontales
  • d-* - Affichage
  • position-* - Positionnement
  • overflow-* - Débordement
  • shadow-* - Ombres
  • border-* - Bordures
