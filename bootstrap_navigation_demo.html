Résumé des Classes Navigation

Navbar
  • navbar - Navbar de base
  • navbar-brand - Marque/logo
  • navbar-nav - Conteneur des liens
  • nav-link - Lien de navigation
  • navbar-toggler - Bouton mobile
  • navbar-expand-* - Responsive
  • bg-* - Couleurs de fond
Navs et autres
  • nav - Navigation de base
  • nav-tabs - Onglets
  • nav-pills - Pills
  • breadcrumb - Fil d'Ariane
  • pagination - Pagination
  • dropdown - Menu déroulant
  • tab-content - Contenu des onglets
