Résumé des Classes Formulaires

Classes de base
  • form-control - Input de base
  • form-select - Select dropdown
  • form-check - Checkbox/Radio
  • form-check-input - Input checkbox/radio
  • form-check-label - Label checkbox/radio
  • form-label - Label de formulaire
  • form-text - Texte d'aide
  • form-floating - Label flottant
États et validation
  • is-valid - État valide
  • is-invalid - État invalide
  • valid-feedback - Message de succès
  • invalid-feedback - Message d'erreur
  • form-check-inline - Checkbox inline
  • form-switch - Switch toggle
  • form-range - Range slider
  • input-group - Groupe d'inputs
